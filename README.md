## A simpler but complete blazor controls library that works.

KLib contains principal controls, based on bootstrap 5.3 and simplifying the life of blazor programmers, higly customizable.
Check the example project for simple startup of a project.
## Components:
- GoBack
- KButton
- KCard2
- KCheckBox
- KCollapsible
- KComboBox
- KDatePicker
- KNumericBox
- KSlider
- KSpinner
- KSplitter
- KSwitch
- KTable
- KTableColumn
- KTextBox
- Tab
- TabContent
- Tabs
